## Usage

### Run the Project

1. Install the dependencies:
    ```
    $ yarn install
    ```

2. Start the project:
    ```
    $ yarn start
    ```

### Preview with Storybook

```
$ yarn storybook
```

### Format with Prettier

```
$ yarn formaat
```

### Lint and Tests

1. Lint (no warnings allowed by default):
    ```
    $ yarn lint
    ```

2. Run all tests:
    ```
    $ yarn test
    ```
